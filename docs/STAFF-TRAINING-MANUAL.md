## 1. Introduction à Temba

### Qu'est-ce que Temba ?

**Temba** est la plateforme de billetterie en ligne leader au Burkina Faso. Elle permet aux utilisateurs de :

- 🎫 **Découvrir** des événements culturels, concerts, festivals et spectacles
- 💳 **Réserver** des billets en ligne de manière sécurisée
- 📲 **Recevoir** des e-billets avec QR code unique
- 🔄 **Transférer** des billets à d'autres personnes
- 📱 **Accéder** à leurs billets hors ligne

### Public Cible

- Amateurs d'événements culturels
- Festivaliers
- Familles recherchant des activités
- Professionnels organisant des sorties d'équipe

### Avantages de Temba

| Avantage | Description |
|----------|-------------|
| **Simplicité** | Interface intuitive en français |
| **Sécurité** | Paiements cryptés et QR codes uniques |
| **Flexibilité** | Transfert de billets possible |
| **Accessibilité** | Application mobile + site web |
| **Local** | Paiement en FCFA, Mobile Money accepté |

## 3. Création de Compte et Connexion

### 3.1 Créer un Nouveau Compte

#### Option A : Inscription par Téléphone (Recommandée)

1. Ouvrez l'application Temba
2. Tapez sur "**Créer un compte**"
3. Sélectionnez "**Téléphone**" comme méthode d'inscription
4. Remplissez les informations :
   - **Nom complet** (ex: Kabore Jean)
   - **Indicatif pays** (+226 pour Burkina Faso)
   - **Numéro de téléphone** (ex: 70 12 34 56)
   - **Mot de passe** (minimum 8 caractères)
   - **Confirmation du mot de passe**
5. Tapez sur "**Créer le compte**"
6. Votre compte est créé !

> **💡 Conseil:** Utilisez le même numéro de téléphone que votre Mobile Money pour faciliter les paiements.

#### Option B : Inscription par Email

1. Ouvrez l'application Temba
2. Tapez sur "**Créer un compte**"
3. Sélectionnez "**Email**" comme méthode d'inscription
4. Remplissez les informations :
   - **Nom complet**
   - **Adresse email** (ex: nom@exemple.com)
   - **Mot de passe** (minimum 8 caractères)
   - **Confirmation du mot de passe**
5. Tapez sur "**Créer le compte**"
6. Votre compte est créé !

### 3.2 Se Connecter

1. Ouvrez l'application Temba
2. Tapez sur "**Se connecter**"
3. Entrez votre **email** ou **numéro de téléphone**
4. Entrez votre **mot de passe**
5. Tapez sur "**Connexion**"

### 3.3 Mot de Passe Oublié

1. Sur l'écran de connexion, tapez sur "**Mot de passe oublié ?**"
2. Entrez votre email ou numéro de téléphone
3. Suivez les instructions envoyées par email/SMS
4. Créez un nouveau mot de passe

### 3.4 Pays Supportés pour l'Inscription

Temba supporte les inscriptions depuis de nombreux pays d'Afrique de l'Ouest et d'ailleurs :

| Région | Pays |
|--------|------|
| **Afrique de l'Ouest** | 🇧🇫 Burkina Faso, 🇨🇮 Côte d'Ivoire, 🇬🇭 Ghana, 🇸🇳 Sénégal, 🇲🇱 Mali, 🇳🇪 Niger, 🇹🇬 Togo, 🇧🇯 Bénin, 🇳🇬 Nigeria |

## 6. Réserver des Billets

### 6.1 Processus de Réservation Étape par Étape

#### Étape 1 : Sélectionner l'Événement

1. Trouvez l'événement souhaité
2. Tapez dessus pour ouvrir la page détails
3. Vérifiez les informations (date, heure, lieu)

#### Étape 2 : Choisir les Billets

1. Dans la section "**Types de Billets**", vous verrez les options disponibles
2. Pour chaque type de billet :
   - Utilisez les boutons **-** et **+** pour ajuster la quantité
   - Le prix unitaire est affiché
   - Certains billets peuvent avoir une limite par commande
3. Le **sous-total** s'affiche en bas

#### Étape 3 : Vérifier le Récapitulatif

Avant de payer, vérifiez :
- ✅ Nombre de billets correct
- ✅ Types de billets corrects
- ✅ Total correct (incluant les frais de traitement de 2%)

#### Étape 4 : Valider la Commande

1. Tapez sur "**Réserver**" ou "**Continuer vers le paiement**"
2. Vous serez dirigé vers la page de paiement

### 6.2 Événements Multi-Dates

Certains événements se déroulent sur plusieurs dates :

1. Sur la page de l'événement, vous verrez une **liste de dates disponibles**
2. **Sélectionnez la date** qui vous convient
3. Les billets disponibles pour cette date s'afficheront
4. Procédez à la réservation normalement

> **⚠️ Important:** Assurez-vous de sélectionner la bonne date avant de réserver !

### 6.3 Récapitulatif du Panier

## 7. Processus de Paiement

### 7.1 Méthodes de Paiement Disponibles

| Méthode | Description | Disponibilité |
|---------|-------------|---------------|
| 📱 **Mobile Money** | Orange Money, Wave, Moov Money | ✅ Disponible (via PawaPay) |
| 💳 **Carte Bancaire** | Visa, Mastercard | ✅ Disponible (via Stripe) |

### 7.2 Paiement par Mobile Money (Recommandé)

Le paiement Mobile Money est traité par **PawaPay** et supporte :

| Fournisseur | Pays |
|-------------|------|
| 🟠 **Orange Money** | Burkina Faso |
| 🔵 **Wave** | Burkina Faso, Sénégal |
| 🟢 **Moov Money** | Burkina Faso |

#### Processus de Paiement Mobile Money

1. Sélectionnez "**Mobile Money**" comme méthode de paiement
2. Choisissez votre **fournisseur** (Orange Money, Wave, ou Moov Money)
3. Entrez votre **numéro de téléphone** (format: +226 XX XX XX XX)
4. Tapez sur "**Payer**"
5. **Deux scénarios possibles :**

   **Scénario A - Redirection automatique :**
   - Vous êtes redirigé vers la page de paiement PawaPay
   - Confirmez le paiement sur votre téléphone
   - Retour automatique vers Temba après confirmation

   **Scénario B - Code OTP requis (Orange Money) :**
   - Un code USSD s'affiche : `*144*4*6*[MONTANT]#`
   - Composez ce code sur votre téléphone
   - Entrez le code OTP reçu par SMS
   - Validez le paiement

### 7.3 Paiement par Carte Bancaire

#### Nouvelle Carte

1. Sélectionnez "**Carte Bancaire**"
2. Entrez les informations de votre carte :
   - **Numéro de carte** (16 chiffres)
   - **Date d'expiration** (MM/AA)
   - **CVV** (3 chiffres au dos de la carte)
   - **Nom du titulaire**
3. Optionnel : Cochez "**Sauvegarder cette carte**" pour les prochains achats
4. Tapez sur "**Payer**"
5. Attendez la confirmation

#### Carte Enregistrée

Si vous avez des cartes enregistrées :
1. Sélectionnez la carte souhaitée dans la liste
2. La carte s'affiche sous forme "•••• •••• •••• 1234"
3. Tapez sur "**Payer**"

### 7.4 Conversion de Devises (Carte uniquement)

Pour les paiements par carte, le montant en FCFA est automatiquement converti :

- Le **taux de change** actuel est affiché
- Le **montant en EUR** (ou autre devise) est indiqué
- Les conversions sont effectuées par Stripe de manière sécurisée

> **💡 Note:** Les paiements Mobile Money sont traités directement en FCFA, sans conversion.

### 7.5 Confirmation de Paiement

Après un paiement réussi :

1. ✅ Vous recevez une **confirmation à l'écran**
2. 📧 Un **email de confirmation** est envoyé
3. 🎫 Vos **billets sont disponibles** dans "Mes Billets"
4. 📲 Vous pouvez **télécharger** vos billets immédiatement

### 7.6 Paiement en tant qu'Invité

Il est possible d'acheter des billets sans créer de compte :

1. Sur la page de paiement, restez sur "**Continuer en tant qu'invité**"
2. Entrez vos informations :
   - **Nom complet**
   - **Email** ou **Téléphone**
3. Procédez au paiement
4. Vos billets seront envoyés par email
5. Un lien de vérification sera fourni pour accéder à vos billets

> **💡 Conseil:** Créer un compte offre plus d'avantages (historique, transferts, notifications).

## 9. Transfert de Billets

### 9.1 Qu'est-ce que le Transfert de Billets ?

Le transfert permet de **donner un billet à quelqu'un d'autre** :

- ✅ Le destinataire reçoit le billet
- ✅ Le billet est retiré de votre compte
- ✅ Le nouveau propriétaire peut utiliser le billet
- ✅ Le transfert est irréversible une fois accepté

### 9.2 Comment Transférer un Billet

#### Étape 1 : Sélectionner le Billet

1. Allez dans **Profil** → **Mes Billets**
2. Trouvez le billet à transférer
3. Tapez sur "**➡️ Transférer**"

#### Étape 2 : Choisir la Méthode de Transfert

Deux options disponibles :

**Option A : Par Email**
1. Sélectionnez "**Email**"
2. Entrez l'**adresse email** du destinataire
3. Optionnel : Entrez le **nom** du destinataire
4. Optionnel : Ajoutez un **message personnel**

**Option B : Par Téléphone**
1. Sélectionnez "**Téléphone**"
2. Entrez le **numéro de téléphone** du destinataire (format: +226XXXXXXXX)
3. Optionnel : Entrez le **nom** du destinataire
4. Optionnel : Ajoutez un **message personnel**

#### Étape 3 : Confirmer le Transfert

1. Vérifiez les informations affichées
2. Tapez sur "**Confirmer le transfert**"
3. Le billet est transféré !

### 9.3 Transfert vers un Utilisateur Non-Inscrit

Si le destinataire n'a pas de compte Temba :

1. Le billet est mis en **"transfert en attente"**
2. Le destinataire reçoit une **notification** (email ou SMS)
3. Quand il crée un compte avec le même email/téléphone, le billet lui est **automatiquement attribué**

### 9.4 Voir les Billets Transférés

#### Billets Reçus (que vous avez reçus)

1. Allez dans **Profil** → **Billets Reçus**
2. Tous les billets que d'autres personnes vous ont transférés s'affichent
3. Ces billets fonctionnent comme vos propres billets

#### Billets Envoyés (que vous avez donnés)

1. Allez dans **Profil** → **Billets Envoyés**
2. L'historique de vos transferts s'affiche :
   - Destinataire
   - Date du transfert
   - Statut (Complété, En attente)

### 9.5 Statuts de Transfert

| Statut | Description |
|--------|-------------|
| ✅ **Complété** | Le destinataire a reçu le billet |
| ⏳ **En attente** | Le destinataire n'a pas encore de compte |
| ❌ **Annulé** | Le transfert a été annulé |

## 12. Questions Fréquentes (FAQ)

### Compte et Connexion

**Q: J'ai oublié mon mot de passe, que faire ?**
> R: Tapez sur "Mot de passe oublié" sur l'écran de connexion et suivez les instructions.

**Q: Puis-je avoir plusieurs comptes ?**
> R: Non, un seul compte par email ou numéro de téléphone est autorisé.

**Q: Comment changer mon numéro de téléphone ?**
> R: Allez dans Profil → Informations du Profil → Modifiez le téléphone.

### Billets

**Q: Mes billets sont-ils valides hors ligne ?**
> R: Oui, les billets dans l'application mobile fonctionnent hors ligne.

**Q: Puis-je annuler un billet ?**
> R: Les politiques d'annulation dépendent de chaque événement. Contactez le support.

**Q: Puis-je transférer un billet déjà scanné ?**
> R: Non, seuls les billets non utilisés peuvent être transférés.

**Q: Le transfert est-il gratuit ?**
> R: Oui, le transfert de billets est entièrement gratuit.

### Paiement

**Q: Quels moyens de paiement acceptez-vous ?**
> R: Nous acceptons le **Mobile Money** (Orange Money, Wave, Moov Money) via PawaPay et les **cartes bancaires** (Visa, Mastercard) via Stripe.

**Q: Le paiement est-il sécurisé ?**
> R: Oui, les paiements Mobile Money sont traités par PawaPay et les paiements par carte par Stripe, deux plateformes sécurisées et certifiées.

**Q: Dans quelle devise sont les prix ?**
> R: Les prix sont affichés en FCFA (Franc CFA d'Afrique de l'Ouest). Les paiements Mobile Money sont traités directement en FCFA.

**Q: Y a-t-il des frais supplémentaires ?**
> R: Oui, des frais de traitement de 2% s'appliquent à chaque commande.

**Q: Comment fonctionne le code OTP pour Orange Money ?**
> R: Pour certains paiements Orange Money, vous devez composer un code USSD (`*144*4*6*[MONTANT]#`) et entrer le code OTP reçu par SMS.

### Événements

**Q: Comment savoir si un événement est annulé ?**
> R: Vous recevrez une notification et un email. Vérifiez également la page de l'événement.

**Q: Puis-je acheter des billets pour quelqu'un d'autre ?**
> R: Oui, achetez les billets puis transférez-les à la personne.

## 13. Résolution des Problèmes

### Problèmes Courants

#### 🔴 "Erreur de connexion"

**Solutions :**
1. Vérifiez votre connexion internet
2. Vérifiez que votre email/téléphone est correct
3. Vérifiez votre mot de passe (sensible à la casse)
4. Essayez de réinitialiser votre mot de passe

#### 🔴 "Erreur de paiement" (Carte)

**Solutions :**
1. Vérifiez que votre carte est valide
2. Vérifiez que vous avez suffisamment de fonds
3. Vérifiez que les informations de carte sont correctes
4. Essayez une autre méthode de paiement (Mobile Money)
5. Contactez votre banque si le problème persiste

#### 🔴 "Erreur de paiement" (Mobile Money)

**Solutions :**
1. Vérifiez que vous avez suffisamment de solde sur votre compte Mobile Money
2. Vérifiez que le numéro de téléphone est correct (format: +226XXXXXXXX)
3. Pour Orange Money : composez le code USSD affiché et entrez le code OTP
4. Attendez quelques secondes et réessayez
5. Essayez avec un autre fournisseur (Wave, Moov Money)
6. Contactez votre opérateur si le problème persiste

#### 🔴 "QR code ne fonctionne pas"

**Solutions :**
1. Assurez-vous que la luminosité de l'écran est au maximum
2. Nettoyez l'écran de votre téléphone
3. Essayez de télécharger le billet en PDF
4. Contactez le support Temba

#### 🔴 "Billet non affiché"

**Solutions :**
1. Tirez vers le bas pour rafraîchir la page
2. Vérifiez votre connexion internet
3. Déconnectez-vous et reconnectez-vous
4. Vérifiez dans "Billets Reçus" si le billet a été transféré

#### 🔴 "Database error saving new user" (Application mobile)

**Solutions :**
1. Vérifiez que vous n'avez pas déjà un compte avec cet email/téléphone
2. Essayez avec un autre email ou numéro de téléphone
3. Attendez quelques minutes et réessayez
4. Contactez le support technique
